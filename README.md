# Hangman Game in Python 

This is a simple * HANGMAN GAME * built using Python.  
It is a beginner-friendly project that demonstrates basic programming concepts such as loops, conditions, functions, sets, and the random module.


## Features

- Random word selection  
- Hangman ASCII art display  
- Input validation  
- Score system : 
  - Start with 100 points  
  - +5 points for each correct guess  
  - -10 points for each wrong guess  
  - Bonus points for winning with fewer wrong guesses  
- Tracks best score  
- Play again option
